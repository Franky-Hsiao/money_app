# money_app

### 簡介
  課餘時間練習android studio所嘗試開發的APP，其主要功能是讓使用者可以輕鬆紀錄生活中的收入支出，並用簡易的圖表讓人可以快速看出消費比例。

### 特點

- 幫助使用者可以簡單的紀錄金錢收支
- 具有多國語言，可供 APP 內切換
- 有圖表讓使用者更能理解金錢狀況
- 可自訂要搜尋的時間範圍
